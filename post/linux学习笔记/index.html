一、文件系统命令

1.shell

shell是Linux里用户使用Linux的桥梁（命令接口），Linux下默认的shell一般是bash。

Linux的通用命令格式：命令 [选项] [参数]

选项：

  • 以“-”引导短格式选项（单个字符）
  • 以“- -”引导长格式选项（多个字符）
  • 多个短格式选项可以写在一起，只用一个“-”引导

常用的shell快捷键：

  • Tab键：自动补齐
  • 快捷键Ctrl＋U：清空至行首
  • 快捷键Ctrl＋L：清屏
  • ↑：上一条历史记录
  • ↓：下一条历史记录

命令帮助文档

sudo yum install coreutils-help bash-help  //安装
[命令] --help  //适用于大多数文档

2.Linux文件系统

Linux文件系统继承了Unix的特点，它采用了==树形目录结构==。

/：根目录
/boot：启动文件
/dev：设备文件
/proc：内核与进程镜像
/mnt：临时挂载
/media：挂载媒体设备
/root：root用户的$HOME目录
/home：包含普通用户的$HOME目录
/bin：系统程序
/sbin：管理员系统程序
/lib：系统程序库文件
/etc：系统程序与大部分应用程序的全局配置文件
/usr：系统存放程序的目录
/var：动态数据（包含log系统日志文件）
/tmp：临时文件

3.目录操作命令

基础操作命令：

pwd //打印当前工作目录的绝对路径
.   //代表当前所在目录
..  //代表当前目录位置的上一层目录
~   //代表家目录
cd  //更改工作目录路径

使用mkdir创建目录：

mkdir test             //在当前工作目录下创建目录test
mkdir -p picture/test  //若路径中的目录不存在，先创建目录

使用rmdir删除空目录。 使用ls列出目录和文件信息：

选项 说明
-l 以详细信息的形式展示出当前目录的文件
-a 显示当前目录中的全部文件（包括以.开头的隐藏文件）
-d 查看目录属性
-t 按照创建时间顺序列出文件
-h 与-l一起，以易于阅读的形式输出文件大小

4.文件操作命令

使用touch创建空文件、更改文件时间。 使用rm删除文件或者目录：

选项 说明（一般使用rm -rf删除目录）
-f 强制删除，无需逐一确认
-r 递归删除目录

使用cp复制文件或目录：

cp etc/passwd ./    //复制passwd文件到当前目录下
cp -a etc/passwd passwd2   //保留链接、文件属性，并复制目录下的所有内容到新命名文件passwd2中

使用mv改名、移动目录或文件：

mv [原文件名] [现文件名]  //改名
mv [原文件] [现目录]     //移动

2.管道

==管道==可以把一系列命令连接起来，这意味着第一个命令的输出会作为第二个命令的输入通过管道传给第二个命令。

  • grep命令可以对输入的内容进行搜索（按Ctrl+D结束输入）。 多个命令使用管道连接： 管道只接受标准输出，不接受标准错误。

三、用户和权限管理

1.用户和组

用户类型分为root用户和普通用户。

root用户拥有Linux系统的最高权限。

/etc/passwd用于保存用户的帐号基本信息。

组：当创建一个新的用户账户时,如果没有指定该用户属于哪一个组,那么Linux就会创建一个和该用户==同名==的组,这个组就是私有组。 也可以为用户指定附加组（任意多个），如==技术主管==，属于技术组，也属于主管组。 /etc/group保存组帐号基本信息。 使用su命令可以在任意用户之间进行切换。不指定用户时，切换到root用户。 以下命令均需要使用root用户执行：

- 添加用户：useradd命令
- 设置用户密码：passwd命令
- 删除用户：userdel命令
          userdel -r （同时删除用户及主目录）
- 添加组：groupadd命令
- 将用户添加到组：usermod命令
               usermod -a -G （指定要将用户添加到哪个组且不覆盖原有组）
- 删除组：groupdel命令

2.sudo

使用root用户时，如果操作不当，很容易对系统造成破坏，如误删文件，因此，建议尽量少地使用root用户，改为使用sudo。

  • 用户通过sudo命令输入自己的密码可以临时获得root的部分权限，可以不必知道root的密码，保证系统的安全性。
  • 用户需要添加sudo权限，才可以使用sudo命令。
  • sudo -i可以临时切换到root终端。

添加sudo权限方法：

  1. 安装openeuler时，创建用户步骤如果勾选了“将些用户做为管理员”，这个用户就会被添加sudo权限；

3.文件和目录权限

  • 可读（r）：允许查看文件内容、显示目录列表
  • 可写（w）：允许修改文件内容，允许在目录中新建、移动、删除文件或子目录
  • 可执行（x）：允许运行程序、切换目录

使用chmod命令设置权限：

chmod  [-R] [ugoa][+-=][rwx]  文件或目录   //字符法

使用-R选项可以递归设置目录的权限。

chown命令可以更改文件和目录的所有者和所属组，需要root权限执行。

chown [-R] [user][:group] 文件或目录

-R选项用来递归修改目录。
